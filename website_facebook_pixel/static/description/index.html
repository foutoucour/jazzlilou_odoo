Configuration

To configure, follow these steps:

  • Go to the Website - Configuration - Settings menu.
  • Activate Facebook Pixel option.
  • Enter Facebook Pixel Key.
  • Apply the changes.
  • The Facebook Pixel will be activated on all pages of the website.
